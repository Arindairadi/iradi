

# Personal Portfolio Website

A modern, polished single-page portfolio website for a Computer Science student and software developer. Clean design with smooth animations, professional typography, and a minimal dark/light color palette.

---

## Sections

### 1. Hero Section
- Large, bold name with animated entrance
- Professional title and tagline
- Two CTA buttons: "View Projects" (scrolls to projects) and "Contact Me" (scrolls to contact)
- Subtle background gradient or pattern

### 2. About Me
- Professional summary paragraph
- Highlights: academic background, passion for technology, interests in web dev, data science, networking, and social impact

### 3. Skills
- Four category cards displayed in a grid:
  - Programming Languages (Python, JavaScript, Java, C/C++)
  - Web Technologies (HTML, CSS, React, Node.js)
  - Tools & Platforms (Git, GitHub)
  - Soft Skills (problem solving, teamwork, research, communication)
- Each skill shown as a badge/tag within its card

### 4. Projects
- Card grid layout (2-3 columns on desktop, 1 on mobile)
- Each card: title, description, tech tags, GitHub link, optional live demo link
- Placeholder projects included, easy to swap out with real data

### 5. Experience & Education
- Timeline-style layout
- Entries for education, internships, volunteering, leadership
- Placeholder content ready to customize

### 6. Certifications & Achievements
- Simple list or card layout for certificates, awards, and notable achievements

### 8. Footer
- Copyright line with name and year
- Social media icon links (GitHub, LinkedIn, Email)

## Design & UX
- **Color palette**: Minimal blue-gray tones, professional and clean
- **Typography**: Modern sans-serif hierarchy
- **Fully responsive**: Mobile-first with breakpoints for tablet and desktop
- **Smooth scrolling** between sections via navigation
- **Subtle fade-in animations** as sections scroll into view
- **Sticky top navigation** with section links and smooth scroll
- **Dark mode support** via theme toggle
